# 🏪 Carone Supermercados - Sistema de Gestão

Sistema de gestão para supermercados desenvolvido com Node.js, PostgreSQL e Docker.

## 🚀 Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Node.js, Express
- **Banco de Dados:** PostgreSQL 15
- **DevOps:** Docker, Docker Compose, Nginx
- **Ferramentas:** Git, PgAdmin

## 📋 Pré-requisitos

- Docker e Docker Compose
- Git
- Node.js 18+ (para desenvolvimento)

## 🎯 Funcionalidades

- ✅ Dashboard com métricas em tempo real
- ✅ Cadastro completo de produtos (CRUD)
- ✅ Controle de estoque com alertas
- ✅ Gestão de preços e margens
- ✅ Relatórios de vendas
- ✅ Interface responsiva
- ✅ Alertas de estoque baixo
